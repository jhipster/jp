# <i class="fa fa-usd"></i> スポンサー

JHipsterは資金を集めるために[OpenCollective](https://opencollective.com/generator-jhipster)を使っています。資金は透明性のある方法でプロジェクトの経費（このウェブサイトの運営など）に充てられます（[OpenCollectiveのドキュメントを参照](https://opencollective.com/how-it-works)）。あなたの寄付は、プロジェクトが存続し、成長するのに役立ちます。

後援者やスポンサーになるには、[OpenCollectiveページ](https://opencollective.com/generator-jhipster)からお願いします。

## 免責事項

これは寄付です。見返りとして商品やサービスは提供されません。そのような目的での返金要求は拒否されます。

弊社は、[規約](/association/)に記載されている通り、フランスの非営利団体であり、ビジネスパートナーシップには関心がありません。

後援者は、月に2ドルをプロジェクトに寄付します。

Open Collective APIを使用して、メインページに後援者を自動的に表示します。

- [https://www.jhipster.tech](https://www.jhipster.tech)のトップページの「後援者」欄に掲載されます（約11万ビュー/月）。
- GitHubのメインプロジェクトページの「後援者」欄に掲載されます（約15,000ビュー/月）。

## プロジェクトはその資金で何をするのでしょうか？

[Open Collectiveのドキュメント](https://docs.opencollective.com)にあるように、あなたのお金がどのように使われるかは公開され、完全に透明性が確保されます。

誰でも経費を申請できます。その経費がコミュニティの発展にとって意味のあるものであれば、コアコントリビューターたちによって私たちのOpen Collectiveの台帳に「マージ」され、経費を申請した人へ償還されます。

資金はまず、プロジェクトの直接経費に充当されます。

- [バグバウンティ](/bug-bounties)
- ウェブサイトホスティング
- デベロッパーツール
- 法的保護

その後、資金は以下の用途にも使われます。

- プロジェクトグッズの支払い
- ミートアップ料金の支払い
- 開発者の集まりのための費用：コアチームの会議のための旅費と食事代

## ヘルプ！スポンサーですが、ロゴが表示されません！

各スポンサーシップレベルについて、Open Collective APIを使用して、メインページにスポンサーを自動的に表示します。そのため、ロゴが表示されない場合は、次の点を確認してください。

- 正しいスポンサーシップレベルを選択していることを確認してください。これは、上記にリストされているレベルのいずれかでなければなりません。カスタム寄付ではありません。
- Open Collectiveプロフィールでロゴがアップロードされていることを確認してください。これは、メインページにロゴを自動的に表示するために使用されるものです。
- 数分待ってから確認してください。Open Collective APIはキャッシュされているため、ロゴが表示されるまでに数分かかります。

これらの手順で解決しない場合は、Open Collectiveサポートにお問い合わせください。弊社は、Open Collective APIを制御できないためです。Open Collective側にバグや技術的な問題がある可能性があります。
