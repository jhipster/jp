# <i class="fa fa-usd"></i> スポンサー

JHipsterは資金を集めるために[OpenCollective](https://opencollective.com/generator-jhipster)を使っています。資金は透明性のある方法でプロジェクトの経費（このウェブサイトの運営など）に充てられます（[OpenCollectiveのドキュメントを参照](https://opencollective.com/how-it-works)）。皆様の寄付は、プロジェクトが存続し、成長するのに役立ちます。

後援者やスポンサーになるには、[OpenCollectiveページ](https://opencollective.com/generator-jhipster)からお願いします。

## 免責事項

これは寄付です。見返りとして商品やサービスは提供されません。そのような目的での返金要求は拒否されます。

私たちは、[ステータス](/association/)に記載されているように、フランスの非営利団体であり、ビジネスパートナーシップには関心ありません。

後援者は、月に2ドルをプロジェクトに寄付します。

Open Collective APIを使用して、メインページに後援者を自動的に表示します。

- [https://www.jhipster.tech](https://www.jhipster.tech)のトップページの「後援者」欄に掲載されます（月間約11万ビュー）。
- GitHubのメインプロジェクトページの「後援者」欄に掲載されます（月間約1万5000ビュー）。

## プロジェクトはその資金で何をするのでしょうか？

[Open Collectiveのドキュメント](https://docs.opencollective.com)にあるように、皆様のお金がどのように使われるかは公開され、完全に透明性が確保されます。

誰でも経費を申請できます。その経費がコミュニティの発展にとって意味のあるものであれば、コアコントリビューターによって私たちのOpen Collectiveの台帳に「マージ」され、経費を申請した人へ償還されます。

資金はまず、プロジェクトの直接経費に充当されます。

- [バグバウンティ](/bug-bounties)
- ウェブサイトホスティング
- 開発者ツール
- 法的保護

その後、資金は以下の用途にも使われます。

- プロジェクトグッズの支払い
- ミートアップ料金の支払い
- 開発者集会の費用：コアチーム会議の旅費と食事代

## ヘルプ！スポンサーなのにロゴが表示されません！

各スポンサーシップレベルについて、Open Collective APIを使用して、メインページにスポンサーを自動的に表示します。そのため、ロゴが表示されない場合は、次のことを確認してください。

- 正しいスポンサーシップレベルを選択していることを確認してください。これは、上記にリストされているレベルのいずれかでなければならず、カスタム寄付ではありません。
- Open Collectiveプロフィールでロゴがアップロードされていることを確認してください。これが、メインページにロゴを自動的に表示するために使用されるものです。
- 数分待ってください。Open Collective APIはキャッシュされているため、ロゴが表示されるまで数分かかります。

これらの手順が機能しない場合は、Open Collectiveサポートにお問い合わせください。Open CollectiveのAPIは私たちが管理できないためです。バグや技術的な問題が発生している可能性があります。

そして、このページの上部にある免責事項を忘れないでください。「これは寄付です。見返りとして商品やサービスは提供されません。そのような目的での返金要求は拒否されます。」
