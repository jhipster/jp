[Microsoft Azure](https://azure.microsoft.com/overview/?WT.mc_id=java-0000-judubois)は、クラウドでJHipsterアプリケーションを実行するための素晴らしいソリューションです。

- 最も簡単な方法は、[JHipster Azure Container Apps](https://github.com/Azure/generator-jhipster-azure-container-apps)を使用してプロジェクトを最初から最後まで作成およびデプロイすることです。JHipster Azure Container Appsは、フルスタックのSpringアプリケーション開発を効率化するように設計されており、[Azure Container Apps](https://learn.microsoft.com/azure/container-apps/java-overview)の堅牢なインフラストラクチャを活用して、前例のない容易さと効率でプロジェクトを実現します。
- Spring Bootを使用している場合は、[Azure Container Apps](https://azure.microsoft.com/products/container-apps)が組み込みのJava機能でJHipsterアプリケーションを完全にサポートします。[クイックスタート](https://learn.microsoft.com/azure/container-apps/java-get-started)で詳細を読むことができます。
- [Azure App Service](https://azure.microsoft.com/services/app-service/?WT.mc_id=online-jhipster-judubois)を使用してモノリスをデプロイすることもできます。
- 他のDockerおよびKubernetesクラウドプロバイダーと同様に、JHipsterのDockerおよびKubernetesサポートを使用して、DockerイメージをMicrosoft Azureにデプロイできます。これらのオプションの詳細については、[Docker Composeのドキュメント](/docker-compose)および[Kubernetesのドキュメント](/kubernetes)を参照してください。

[![Microsoft Azure](/images/logo/logo-azure.png)](https://azure.microsoft.com/overview/?WT.mc_id=java-0000-judubois)


## NubesGenを使用したデプロイ

[NubesGen](https://www.nubesgen.com)は、JHipsterアプリケーションを完全にサポートするデプロイプラットフォームであり、アプリケーションをAzureにデプロイするための最良のソリューションです。

NubesGenでJHipsterアプリケーションをデプロイするには、ドロップダウンメニューで以下を選択してください。

- MavenでJHipsterアプリケーションを実行するには、"Spring Boot (Java) with Maven" を選択します。
- GradleでJHipsterアプリケーションを実行するには、"Spring Boot (Java) with Gradle" を選択します。
- DockerでJHipsterアプリケーションをパッケージ化して実行するには、"Docker with Spring Boot" を選択します。パフォーマンスはMavenやGradleよりも優れている可能性がありますが、監視オプションが少なく、MicrosoftからのJavaサポートはありません。

完全なドキュメントについては、[NubesGenのウェブサイト](https://www.nubesgen.com)にアクセスしてください。


## start.jhipster.techで生成されたazdを使用したデプロイ

また、[start.jhipster.tech/generate-azure-application](https://start.jhipster.tech/generate-azure-application)を介して、Azure Containers Appsへのazdデプロイが組み込まれた新しいJHipsterアプリケーションを作成することもできます。これにより、デプロイテンプレートを活用し、「azd up」という1つの簡単なコマンドでデプロイできるオープンソースツールキットである[Azure Development CLI](https://learn.microsoft.com/azure/developer/azure-developer-cli/?WT.mc_id=java-0000-sakriema)を活用できます。
