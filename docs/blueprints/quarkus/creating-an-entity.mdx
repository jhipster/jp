# エンティティの作成

## はじめに

JHipster Quarkusエンティティの作成により以下の生成が可能です。

* データベーステーブル
* Liquibaseチェンジセット
* JPAエンティティ
* Quarkus Panache Active Recordまたはリポジトリ
* 基本的なCRUDを操作するResteasyコントローラ
* データ転送オブジェクト（DTO）
* ページ区切り
* テスト

クライアント側のコンポーネント生成はJHipsterに委任します。
